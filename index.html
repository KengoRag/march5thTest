What Are Wireframes?

Wireframes are drawings/sketches you make of your idea of a website. This drawing helps you visualize your website before programming. We use them because having a drawn representation of what you want makes writing your code easier.

Git Workflow

Git workflow helps when working in groups. Each team member is assigned a branch for them to push their work to. Then, the group leader can decide which branches get merged with the master branch. Then, everyone in the group can pull the data from the master branch. This workflow allows for everyone to be on the same page by pulling from the master. This workflow is also useful because it allows a leader to decide which code to merge with the master and which ones not to.

Ways to Create responsive Web Design

  • Design for mobile first
  • Set content to device width
  • Use javascript to run different chunks of style code based on the size of the user's screen.
  • Use percentages rather than pixels to style and distance elements on your website so that no matter the size of the screen, everything is in its proportiantely proper sizes.

Arrays and Loops

Arrays are varbiables that can contain multiple values inside of them. Loops make code run based on whether or not the requirments you set for the code to run are met.
